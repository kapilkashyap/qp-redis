Property: triggerOn (default is @)
When the character specified using this property (followed by at least another character, say @a) is encountered in the input area an ajax call is made to the specified url. What ever response is got is converted into a paginated drop down so that the user can select from it.
Demo: default behaviour
Sample Code:
$( "#x-demo-1" ).quickpost({
	url: "assets/cricketers.json",
	valueProperty: "name",
	itemTemplate: "<div title='{@:email}'><div>{@:name}</div></div>",
	filterJSON: {startsWith: true}
});
Demo: Setting triggerOn to #
Sample Code:
$( "#x-demo-2" ).quickpost({
	url: "assets/cricketers.json",
	valueProperty: "name",
	itemTemplate: "<div title='{@:email}'><div>{@:name}</div></div>",
	filterJSON: {startsWith: true},
	triggerOn: "#"
});
