Property: dropdownWidth (default is null)
This property is used to set the width of the drop down container. If not specified, the width of the drop down container is set to the width of the bound element.
Demo: default behaviour
Sample Code:
$( "#x-demo-1" ).quickpost({
	url: "assets/cricketers.json",
	valueProperty: "name",
	itemTemplate: "<div title='{@:email}'><div>{@name}</div></div>",
	filterJSON: {startsWith:true}
});
Demo: Setting dropdownWidth to 200
Sample Code:
$( "#x-demo-2" ).quickpost({
	url: "assets/cricketers.json",
	valueProperty: "name",
	itemTemplate: "<div title='{@:email}'><div>{@name}</div></div>",
	dropdownWidth: 200,
	filterJSON: {startsWith:true}
});
