Property: closeBtnRequired (default is true)
This property allows you to have a close button displayed when the drop down is shown. However, you can override this property and set it to false if you do not want the close button.
#Pressing the Esc key or blurring the input element will also close the drop down.
Demo:
Sample Code:
$( "#x-demo-1" ).quickpost({
	url: "assets/cricketers.json",
	valueProperty: "name",
	itemTemplate: "<div title='{@:email}'><div>{@:name}</div></div>",
	closeBtnRequired: false,
	filterJSON: {startsWith:true}
});
