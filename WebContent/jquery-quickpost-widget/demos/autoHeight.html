Property: autoHeight (default is true)
This property specifies if the input area will increase or decrease it's height depending on the contents.
Demo: default behaviour
Sample Code:
$( "#x-demo-1" ).quickpost({
	url: "assets/cricketers.json",
	valueProperty: "name",
	itemTemplate: "<div title='{@:email}'><div>{@:name}</div></div>",
	filterJSON: {startsWith:true}
});
Demo: Setting autoHeight to false
Sample Code:
$( "#x-demo-2" ).quickpost({
	url: "assets/cricketers.json",
	valueProperty: "name",
	itemTemplate: "<div title='{@:email}'><div>{@:name}</div></div>",
	filterJSON: {startsWith:true},
	autoHeight: false
});
