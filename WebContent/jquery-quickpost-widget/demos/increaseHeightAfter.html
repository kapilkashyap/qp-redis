Property: increaseHeightAfter (default is 75 characters)
This property specifies the number of characters after which the height of the input area needs to be increased. After every addition or subtraction of x characters the input area height would be increased or decreased.
#This property works only if autoHeight property is set to true.
Demo: default behaviour
Sample Code:
$( "#x-demo-1" ).quickpost({
	url: "assets/cricketers.json",
	valueProperty: "name",
	itemTemplate: "<div title='{@:email}'><div>{@:name}</div></div>",
	filterJSON: {startsWith:true}
});
Demo: Setting increaseHeightAfter to 50
Sample Code:
$( "#x-demo-2" ).quickpost({
	url: "assets/cricketers.json",
	valueProperty: "name",
	itemTemplate: "<div title='{@:email}'><div>{@:name}</div></div>",
	filterJSON: {startsWith:true},
	increaseHeightAfter: 50
});
