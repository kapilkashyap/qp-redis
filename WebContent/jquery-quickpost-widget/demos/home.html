jQuery UI Quick Post Widget
Quick Post Widget is a Facebook's Update status or Google+'s What's new... input area like widget. This Widget allows you to select profile(s) or page(s) from suggested drop down list, highlights them while you continue entering content and when you post the content the back-end code can use the hidden field's value to construct a direct link to the selected profile(s) or page(s). See identifierProperty on the left navigation for further details.

This widget extends the Paginated Drop Down Widget and has all of its features as well as other specific features like specifying the trigger character, auto height increase or decrease, increase height after certain number of characters and more.

This Widget has dependencies on jQuery-1.6+, jQuery UI Widget Factory 1.8+, jQuery Paginated Drop Down Widget and jQuery filterJSON plugin (also developed by me). The filterJSON plugin is a utility which allows you to filter the JSON based on key (or property) alone. You can explore filterJSON over here. However, using filterJSON plugin is not mandatory and can be completely ignored.

Supports IE7+, FF 3.6+, Opera 12+, Safari 5+ and Chrome.

You can explore all the features and individual properties of this widget using the navigation menu.
